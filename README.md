# 🌿 PantrySave

**Turning surplus into shared value**

---

## 🖥️ Live Prototype

(https://thepantrysave.lovable.app)

## 📌 Project Overview
PantrySave is a location-based web platform designed to reduce food waste while improving access to affordable food.

It enables users to list and sell surplus food items, while others can browse, reserve, and communicate directly with sellers within their community.

---

## 🚀 Key Features

- 📦 List items by **quantity or weight**
- 📍 Location-based food discovery
- 💬 Real-time messaging between buyers and sellers
- 🛒 Cart system with **dynamic total calculation**
- 🎯 Points & rewards system
- 🚚 Flexible delivery options (Pickup / Grab / Lalamove)
